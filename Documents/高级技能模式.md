## 核心模式速览（HPB 对应）

| BF 旧模式 | HPB 新模式 | 说明 |
|---|---|---|
| `ClapStateVar<T>` | 不变 | 跨回合状态包装器 |
| `WriteFree(action, canMove)` | `WriteCompileTime(Action<Community>)` | 编译时自由动作 |
| `WithFree(AttackStage, Action<...>)` | `WithRuntime(AttackStage.CEValue, string analyzerKey)` | 攻击阶段钩子 |
| `WriteEffect(..., duration, action)` | `WriteEffect(..., ClapRoundClock entityClock, string analyzerKey)` | 效果写入 |
| `DefenseBase.Work()` | `DefenseEntity` + `[IsAnalyzer(AnalyzerType.Defense)]` | 自定义防御 |
| `Compile(judger)` | `Compile(JudgeRuleManager?)` | 编译参数 |
| `ICallbackOnJudge.JudgeRule` 委托 | `ICallbackOnJudge.AnalyzerKey` + `IsPlayer` | 动态注册 |
